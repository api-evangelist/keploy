# Keploy

Open-source, AI-native testing platform that captures real production API traffic with eBPF and replays it in CI as deterministic tests, auto-generated mocks, and production-like sandboxes with zero code changes. Offers a Cloud REST API, a hosted MCP server, llms.txt, and generated agent skills.

- **Provider:** https://keploy.io

## APIs

- **Keploy Cloud API** — REST API for Keploy Cloud providing test generation and execution capabilities, app/test-suite management, recordings/mocks, test reports, schema coverage, API keys, clusters, and account/subscription info. Authenticated via scoped API keys (KEPLOY_API_KEY). Base URL confirmed via third-party MCP registry rather than a loadable page on Keploy's own site. (`https://api.keploy.io/client/v1`)
- **Keploy MCP Server** — Hosted, remote Model Context Protocol server (streamable HTTP transport, bearer-token auth, ~80 tools) that proxies Keploy's Cloud REST API, giving agents like Cursor, GitHub Copilot, Claude Code, and Windsurf access to test generation and execution capabilities. Can also emit agent-discovery artifacts including a Claude Code SKILL.md. (`https://api.keploy.io/client/v1/mcp`)

> Keploy exposes a hosted MCP server at https://api.keploy.io/client/v1/mcp (streamable HTTP, bearer-token auth) referenced from its own llms.txt, plus llms.txt and llms-full.txt, and on-demand generated Claude Code agent skills (.claude/skills/keploy/SKILL.md via the devloop_install_repo_artifacts tool). A REST API (base https://api.keploy.io/client/v1) exists but its base URL/paths are confirmed from a third-party verified MCP registry rather than a loadable page on Keploy's own developer site. No static, publicly downloadable OpenAPI/AsyncAPI/GraphQL/Postman contract exists — machine-readable schemas are generated dynamically per-app behind auth via the getGeneratedSchema endpoint. A local CLI (curl -O -L https://keploy.io/install.sh) is available for self-hosted use. Core OSS is Apache-2.0 with commercial Cloud/Enterprise tiers.
